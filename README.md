# 📊 Brazilian Stock Market Multi-Asset Analysis

## 🇧🇷 Português

Este projeto realiza uma análise completa do mercado de ações brasileiro, utilizando múltiplos ativos e o índice IBOV.

O objetivo é estudar comportamento de preços, retornos históricos e tendências de longo prazo.

---

### 📊 Ativos analisados

- EGIE3 (Engie Brasil)
- ITSA4 (Itaúsa)
- BBAS3 (Banco do Brasil)
- PSSA3 (Porto Seguro)
- IBOVESPA (^BVSP)

---

### 📈 Análises realizadas

- Retornos mensais
- Retornos anuais
- Percentual de anos positivos/negativos
- Médias históricas mensais
- Estratégia de médias móveis (20 e 200 dias)

---

### 🛠️ Tecnologias utilizadas

- Python
- Pandas
- Matplotlib
- yfinance

---

### 🎯 Objetivo

Este projeto foi desenvolvido para praticar análise de dados financeiros, estatística descritiva e estratégias básicas de tendência no mercado de ações.

---

## 🇺🇸 English

This project performs a complete analysis of the Brazilian stock market using multiple assets and the IBOV index.

The goal is to study price behavior, historical returns, and long-term trends.

---

### 📊 Assets analyzed

- EGIE3
- ITSA4
- BBAS3
- PSSA3
- IBOVESPA (^BVSP)

---

### 📈 Analysis performed

- Monthly returns
- Annual returns
- Positive/negative year distribution
- Historical monthly averages
- Moving average strategy (20 and 200 days)

---

### 🛠️ Technologies used

- Python
- Pandas
- Matplotlib
- yfinance

---

### 🎯 Purpose

This project was built to practice financial data analysis, descriptive statistics, and basic trend-following strategies in the stock market.
